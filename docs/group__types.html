◆ BoundingBox

using eq::BoundingBox = typedef glm::ivec4

BoundingBox struct composed of 4 components:

  • x
  • y
  • z
  • w

Each component is a public member, an int in the range [-2147483648, 2147483647]. Can be constructed and manipulated very easily:

eq::BoundingBox bb{1, 2, 3, 4} // x = 1, y = 2, z = 3, w = 4
bb.x = 0; // x = 0, y = 2, z = 3, w = 4
bb.w = 0; // x = 0, y = 2, z = 3, w = 0
